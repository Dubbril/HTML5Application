ใน HTML5 ได้ยกเลิกแอตทริบิวต์ align แล้ว

ใน HTML เวอร์ชันก่อน เราสามารถใข้แอตทริบิวต์ align สำหรับกำหนดว่า จะจัดข้อความ และแท็ก <center> สำหรับกำหนดให้ข้อความอยู่กึ่งกลาง ของเพจ แต่ใน HTML5 นั้น ได้ยกเลิกทั้งแอตทริบิวต์ align และแท็ก <center> แล้ว โดยให้เปลี่ยนมาจัดแนวข้อความด้วย พร็อปเพอร์ตี้ text-align ของ CSS แทน
