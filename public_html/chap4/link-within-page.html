HTML - CSS - jQuery

HTML

HyperText Markup Language(HTML) is the main markup language
for displaying web pages and other information that can be
displayed in a web browser.

CSS

Cascading Style Sheets(CSS) is a style sheet language used for
describing the presentation semantics (the look and formatting)
of a document written in a markup language.

jQuery

jQuery is a cross-browser JavaScript library designed to simplify
the client-side scripting of HTML.
