ใน HTML5 ได้ยกเลิกแอตทริบิวต์ align แล้ว

ใน HTML เวอร์ชั่นก่อน เราสามารถใช้แอตทริบิวต์ align สำหรับกำหนดว่าจะจัดข้อความชิดด้านใด และแท็ก <center> สำหรับกำหนดให้ข้อความอยู่กึ่งกลางของเพจ แต่ใน HTML5 นั้นได้ยกเลิกทั้งแอตทริบิวต์ align และแท็ก <center> แล้ว โดยให้เปลี่ยนมาจัดแนวข้อความด้วยพร็อปเพอร์ตี้ text-align ของ CSS แทน
