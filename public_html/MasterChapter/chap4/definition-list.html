เทคโนโลยีที่เกี่ยวข้องกับเว็บไซต์

HTML
ย่อมาจาก HyperText Markup Language เป็นภาษาที่ใช้ในการกำหนด โครงร่างของเอกสารที่จะแสดงผลด้วยเว็บเบราเซอร์

CSS
ย่อมาจาก Cascading Style Sheets เป็นภาษาที่ใช้ในกำหนดรูปแบบหรือ ตกแต่งการเอกสารเว็บเพจ โดยต้องใช้ร่วมกับภาษา HTML

JavaScript
เป็นภาษาที่ใช้ในการเขียนสคริปต์ เพื่อควบคุมการทำงานของ เว็บเพจให้มีการตอบสนองแบบไดนามิก โดยใช้ร่วมกับ HTML และ CSS
